# What you can do with WebRTC

Potencial Use Cases

- File Sharing
- Screensharing
- Video Conferencing / Telepresence
- Integrated text chat
- Virtual conferences
- VOIP Calls
- Remote Desktop applications
- Multiplayer games
- Facial recognition
- Bitmap effects
- Image capture
- Video recording
